# Gesture 3D Web

Browser-based 3D gesture recognition and interaction system using WebGL and machine learning for real-time hand tracking in 3D space.

## Features

- 3D hand tracking in the browser
- Real-time gesture classification
- WebGL-rendered 3D hand model
- Custom gesture training interface
- Cross-browser compatibility

## Tech Stack

TypeScript, WebGL, TensorFlow.js, MediaPipe, Three.js

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/karthik-idikuda/gesture-3d-web.git
cd gesture-3d-web
npm install
```

### Usage

```bash
npm run dev
```
